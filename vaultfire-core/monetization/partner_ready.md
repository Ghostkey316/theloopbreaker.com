# Partner Integration Guide

Vaultfire Core welcomes partnerships that respect the Ghostkey Alignment Code v2.0 and the Ghostkey Ethics Framework v1.0.

* Use provided hooks only when `ethics_anchor` is `true` in `vaultfire_config.json`.
* All integrations must pass an ethics review before deployment.
* Partners may access behavioral yield data via secure API endpoints.
* Sharing or selling user data outside the approved scope is prohibited.
* Failure to comply with ethical standards triggers immediate shutdown as outlined in `../ethics/shutdown_protocol.md`.
