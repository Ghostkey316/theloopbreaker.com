# Vaultfire Design DNA

This document captures the cultural DNA that shapes every user-facing part of Vaultfire.

## 90s-Inspired Moodboard
- **Palette**: bright teals, neon pinks, deep purples.
- **Fonts**: monospace headings with bold drop shadows.
- **Visual cues**: pixel patterns, rounded buttons, playful animations.
- Reference early web design but keep accessibility in mind.

## Ethics-First Branding
- The brand should immediately convey trust and transparency.
- Use the Ghostkey Ethics Framework as the foundation for all imagery and copy.
- Avoid manipulative patterns or dark UX tricks.

## Simple UX
- Pages load quickly and contain only the essentials.
- Buttons speak plain language: “Start,” “Confirm,” “Back.”
- Keep navigation obvious—no hidden controls or surprise flows.

## Human Messages
- Write with warmth and honesty. Examples:
  - “We’re glad you’re here.”
  - “Let us know if something doesn’t feel right.”
  - “Thanks for helping us build Vaultfire.”
- Let the interface feel like a conversation, not a command line.

Stick to these principles whenever new screens or components are added.
