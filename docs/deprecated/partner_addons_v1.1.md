# Partner Add-ons v1.1

This update introduces four optional modules for partners. Each module can be enabled or disabled per partner without affecting core protocol logic.

## 1. Legal & Compliance Layer
- View smart contracts directly from the repository.
- Auto-generate basic Terms of Service and Privacy Policy text using stored region settings.
- Region modes: EU (GDPR), CA (CCPA) and GlobalSafe.
- Content is adapted or restricted based on the configured region.

## 2. Multi-Wallet & Multi-Identity Support
- Link multiple wallets to a single contributor profile.
- Assign ENS names, Twitter handles and custom tags.
- Provides aggregated analytics of wallet and handle counts.

## 3. Dynamic KPI Tracker
- Records engagement and reward events for partner dashboards.
- Generates a summary of DAU, retention, referrals, rewards, passive usage and signal compass activations.
- Includes a redaction toggle to anonymize user stats.

## 4. Automated Partner Mode
- Creates a temporary test user with synthetic data.
- Simulates wallet connects, passive yield, quiz usage and drop claims.
- Guides partners through a short walkthrough and automatically cleans up after 48 hours.

All modules respect the existing privacy layer, ethics rules and loyalty logic. Logs are written through the Verifiability Console for transparency.

## Disclaimers
- Use at your own risk; uptime or results are not guaranteed.
- Ambient data is logged only with opt-in consent.
- Nothing here constitutes legal, medical, or financial advice.

## Ethics-Driven Sandbox Mode
Vaultfire currently operates in sandbox mode to ensure no unintended token distribution or compliance risks. Rewards use test tokens only. Mainnet-ready modules are planned post-audit.
