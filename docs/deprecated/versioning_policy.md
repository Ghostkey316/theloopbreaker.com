# Versioning Policy

Vaultfire modules follow semantic versioning.

- **Major** versions introduce breaking changes.
- **Minor** versions add backward-compatible functionality.
- **Patch** versions fix bugs without changing the API.

Finalized modules begin at `v1.0.0`. Updates increment the minor or patch
level depending on the scope of change.
