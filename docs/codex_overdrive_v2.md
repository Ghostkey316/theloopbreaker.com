# Codex Overdrive V2

Codex Overdrive V2 extends the Vaultfire protocol with adaptive logic.
All core functions now persist data to the repository so progress can be
tracked over time. The implementation lives in
`engine/codex_overdrive_v2.py` and new modules are referenced from the
engine package.

## Features
- **Moral Intelligence Core v2** – checks text for hype and logs each
  decision review.
- **Life Mirror Engine** – creates a personal timeline for users with
  optional emotion tracking.
- **Ambient XP Sync** – issues passive XP into `user_scorecard.json`.
- **Social Alchemy System** – tracks karma points awarded for helping
  others.
- **Learning Forge Mode** – upgrades a user's level when quiz scores are
  high enough.
- **Codex Self-Healing Layer** – confirms key dashboard files exist.
- **Legacy Tracker** – archives notes about meaningful growth sessions.

-## Disclaimers
- Overdrive components are production-ready and subject to change.
- Ambient data collection must comply with user privacy preferences.
